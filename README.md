# Introduction

In today’s digital age, life hacks and viral tips spread rapidly across social media platforms, blogs, and video-sharing sites. While some of these hacks offer practical and time-saving solutions, many are misleading, ineffective, or even unsafe. Users often struggle to distinguish between genuinely useful advice and internet myths that gain popularity simply because they are entertaining or widely shared.  

To address this challenge, **Life Hack or Urban Myth?** was developed as a native Android application that educates users through an interactive flashcard-based quiz system. The app is designed to test users’ critical thinking by presenting life-hack scenarios and asking them to determine whether each one is a valid shortcut or merely an urban myth.  

The project combines education, entertainment, and digital literacy into one engaging platform. Built using **Kotlin** in **Android Studio**, the app also integrates **GitHub** for version control and **GitHub Actions** for automated workflows, ensuring efficient software development practices.  

# Purpose of the Application

The primary purpose of **Life Hack or Urban Myth?** is to help users make informed decisions when encountering viral life hacks online. The app encourages scepticism, evidence-based reasoning, and practical thinking.
